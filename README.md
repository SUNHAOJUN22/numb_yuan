## 🎨 视觉设计理念 (Core Design Aesthetics)

- **极致现代网格 (Minimal Checkered Board)**：去除泥土色与累赘的圆角阴影，改用纯粹平平无奇的极简设计，高分辨率屏幕下无边框自适应，极大提升空间感与阅读清晰度。
- **高刷重构引擎 (High-Refresh Rate Optimizations)**：移除沉重的 CSS 繁复阴影和动效参数，结合 Framer Motion 对 DOM 进行物理级动画渲染引擎重构，专为 144Hz 流畅无掉帧护航。
- **大师级排版组合**：纯净无衬线体系（San-Serif），丢掉冗余的衬线旧元素，**Outfit** 展出级大字标题合集，**JetBrains Mono** 极客数据读数。

完全由 Vite + React 构建，内嵌 Web Audio API 和云端解算通道，无阻塞极速秒开。双语（**简体中文**, **English**）无缝自适应切换。

## 🛠️ 纯前台架构与技术栈 (Technology Stack & Architecture)

- **Frontend Core**: React 18 / TypeScript / Vite构建现代化单页应用。
- **Styling Architecture**: Tailwind CSS 配合极致的响应式与无神仙化设计 (No-slop layout)。
- **Animation Engine**: `framer-motion` (Motion for React) 提供 60FPS 的弹射和微小入场动画体验。
- **Data & Auth Core**: Firebase Firestore (云端极速拉取) + Web LocalStorage (即时防拆) 弹性混合存贮。
- **Audio & Media**: Web Speech API (原声 TTS) 配合 Web Audio API 纯合成弦音，无杂质发声。

# 3. 构建华宇级学位部署镜像
npm run build
