# 袁世杰闹麻了 (Yuan Shijie Nao Ma Le)
### 极简高雅的谷歌风扫雷游戏 • Minimalist Google-Style Minesweeper Game

一个全面、快速、体验流畅的经典扫雷游戏，完美复刻并升级了**谷歌扫雷 (Google Minesweeper)** 的经典极简主义交互界面（Clean Minimalism）。本版本进行了别具一格的幽默化定制：**当你不幸踩到雷时，地底下埋藏的并不是普通的炸弹，而是大大的汉字 “袁”！**

---

## 🎨 视觉设计理念 (Core Design Aesthetics)

这款游戏基于 **Clean Minimalism（极致洁净的现代极简主义）** 主题进行精心打磨：
- **谷歌经典配色系统**：标题与界面细节采用高饱和度、高对比度的 Google 四色（红、蓝、黄、绿）搭配，充满生命力。
- **现代化无内边框格子**：扫雷棋盘去除了传统 Windows 95 泥土色老旧边框，改用轻质淡蓝、浅灰相间的精致格子（Checkered Board），触感灵敏。
- **动态呼吸光效与字体**：使用 **Outfit** 展示级无衬线大字标题，搭配英俊的 **JetBrains Mono** 列印数字计时，视觉节奏极为高雅。
- **无侵入声效反馈**：基于 **Web Audio API** 纯合成正弦波与三角波，实时生成清脆的揭开声、插旗 pop 声及低吟的“袁”式触雷合奏声（100% 离线，无静态音乐包压力）。

## 🎮 玩法与独创操作 (Gameplay & Pro Controls)

1. **安全第一击 (Guaranteed First-Click Safety)**
   - 完美承袭谷歌扫雷演算法，不论你点击哪个网格作为第一步，**首击必然 100% 绝对安全**，并且必定为您自动绽放并消除周围连续的 `3x3` 空白区域，避免开局秒毙！
2. **挖掘模式与插旗模式 (Double Tools)**
   - **掘土（Unearth - Shovel）**：左键单击或直接点按。安全排除并查看数字。
   - **标旗（Place Flags - Flag）**：右键单击，或在下方工具栏切换到插旗模式下点按。将小网格冻结，免受误触。
3. **闪电快捷键 (Keyboard Shortcuts & Reverse Modifiers)**
   - ⌨️ 键盘按 **`F`** 键：可以**一键切换或临时反转**激活的鼠标左键功能（从铲子直接切换到红旗，或反之），无需繁琐地去底部点按，实现电竞玩家的行云流水！
   - ⌨️ 键盘按 **`R`** 键：秒速重新开始当前难度的全新草原！
   - ⌨️ 键盘按 **`ESC`** 键：瞬间关闭弹窗面板。
4. **急速和弦双击 (Pro Chording Click)**
   - 如果某个格子已被揭示（如显示数字 `2`），且其**周围已被你标上了正确数量的红旗（2面旗）**。此时**双击**该格子或对其进行点击，即可**瞬间自动挖掘揭开周围其余所有未插旗的格子**！这是让通关时间压缩进 15 秒内的终极核心秘诀。
